Join us for an immersive workshop where you'll learn how to perform quantification and post-processing of single-cell RNA sequencing (scRNA-seq) data using the powerful Seurat package in R.

Workshop highlights

Comprehensive Quantification Techniques

♣   Learn the fundamentals of scRNA-seq data quantification.
♣   Understand how to preprocess raw sequencing data for analysis.

Mastering Seurat for Post-Processing

♣   Dive deep into Seurat's functionalities for data visualization and interpretation.
♣   Perform clustering, differential expression analysis, and trajectory inference.

Hands-On Practical Sessions

♣   Work with real scRNA-seq datasets.
♣   Gain experience through guided exercises.

Best Practices and Troubleshooting

♣   Discover tips to optimize your analyses
♣   Learn how to address common challenges in scRNA-seq data processing

Intended audience

♣   PIs and trainees wanting to or working on single-cell RNAseq
♣   Anyone interested in the latest techniques for single-cell data analysis.

Prerequisites

♣   Familiarity with genomic concepts is helpful but not required.
♣   Familiarity with the Linux command line is helpful but not required.
♣   Basic knowledge of R programming is recommended.
